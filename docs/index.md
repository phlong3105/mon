## Installation

```shell
git clone https://phlong3105@github.com/phlong3105/mon
cd mon
chmod +x install.sh

# On Linux
conda init bash
bash -i install.sh

# On Mac
conda init zsh
zsh -i install.sh
```

The code is fully compatible with [PyTorch](https://pytorch.org/) >= 2.0.
